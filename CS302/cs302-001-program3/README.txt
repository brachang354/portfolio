Bradley Chang CS302 Program #3 

Ice Cream Tracker README file.

This program will keep track of the various ice cream dessert products for an ice cream shop.

It can keep track of ice cream sandwiches, ice cream cakes, and ice cream pancakes.

Upon booting up the program there'll be a main menu.

The main menu will show the various options to pick, from quitting the program to accessing different dessert categories to displaying currently stored items.

When a dessert category is chosen from the main menu it load another menu to pick options from.
